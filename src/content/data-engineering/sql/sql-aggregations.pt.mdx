## Agregações e GROUP BY

Agregações convertem milhões de linhas em métricas resumidas. `GROUP BY`
torna isso possível.

### Funções básicas

```sql
SELECT
  COUNT(*) AS rows_count,
  COUNT(DISTINCT user_id) AS unique_users,
  SUM(amount) AS total_revenue,
  AVG(amount) AS avg_ticket,
  MIN(amount) AS smallest_sale,
  MAX(amount) AS biggest_sale
FROM payments
WHERE status = 'paid';
```

`COUNT(*)` conta linhas; `COUNT(col)` conta onde col não é NULL;
`COUNT(DISTINCT col)` conta valores únicos não-nulos. Confunda e o
relatório quebra.

### GROUP BY

```sql
SELECT country, COUNT(*) AS orders, SUM(amount) AS revenue
FROM payments GROUP BY country;
```

Colunas adicionais no SELECT precisam estar no GROUP BY (ou ser
agregações):

```sql
-- FALHA:
SELECT country, city, COUNT(*) FROM payments GROUP BY country;
-- OK:
SELECT country, city, COUNT(*) FROM payments GROUP BY country, city;
```

### Ordem lógica revisada

<Diagram type="mermaid">
{`flowchart LR
  A[FROM] --> B[WHERE] --> C[GROUP BY] --> D[HAVING] --> E[SELECT] --> F[ORDER BY] --> G[LIMIT]`}
</Diagram>

`WHERE` filtra antes de agrupar; `HAVING` filtra depois.

```sql
SELECT country, COUNT(*) AS n
FROM payments
WHERE status = 'paid'
GROUP BY country
HAVING COUNT(*) > 100;
```

### Múltiplos níveis

```sql
SELECT
  date_trunc('month', created_at) AS month,
  country,
  COUNT(*), SUM(amount), AVG(amount)
FROM payments
WHERE status = 'paid'
GROUP BY 1, 2
ORDER BY 1 DESC;
```

### Métricas comuns

```sql
-- DAU
SELECT date_trunc('day', event_at), COUNT(DISTINCT user_id)
FROM events GROUP BY 1;

-- AOV
SELECT AVG(amount) FROM orders WHERE status = 'completed';
```

### Erros comuns

- **Esquecer colunas no GROUP BY**.
- **`COUNT(col)` quando queria `COUNT(*)`** — NULLs reduzem.
- **Divisão inteira** sem cast.
- **Filtrar agregações em `WHERE`** — use `HAVING`.
- **`AVG` com outliers**: considere mediana.
- **`COUNT(DISTINCT)` em tabelas enormes**: caro. `APPROX_COUNT_DISTINCT`
  para aproximação.

## Exercício prático

<Exercise
  id="sql-aggregations-exercise"
  title="GROUP BY em JS"
  description="Implemente `groupBy(rows, key, aggregator)`."
  language="javascript"
  starterCode={`function groupBy(rows, key, aggregator) {
  return [];
}

const data = [
  {country:'CO', amount:10},
  {country:'CO', amount:20},
  {country:'MX', amount:15},
];
console.log(groupBy(data, 'country', (rs) => rs.reduce((s,r)=>s+r.amount,0)));`}
  solution={`function groupBy(rows, key, aggregator) {
  const buckets = {};
  for (const r of rows) {
    const k = r[key];
    if (!buckets[k]) buckets[k] = [];
    buckets[k].push(r);
  }
  return Object.entries(buckets).map(([k, group]) => ({
    [key]: k,
    value: aggregator(group),
  }));
}
const data = [
  {country:'CO', amount:10},
  {country:'CO', amount:20},
  {country:'MX', amount:15},
];
console.log(groupBy(data, 'country', (rs) => rs.reduce((s,r)=>s+r.amount,0)));`}
  hints={[
    'Use objeto como dicionário.',
    'Object.entries.map para converter no final.',
  ]}
  tests={[
    {
      description: 'soma por grupo',
      assert: 'groupBy([{c:"a",v:1},{c:"a",v:2}],"c",rs=>rs.reduce((s,r)=>s+r.v,0))[0].value === 3',
    },
    {
      description: 'conta por grupo',
      assert: 'groupBy([{c:"a"},{c:"a"},{c:"b"}],"c",rs=>rs.length).length === 2',
    },
    {
      description: 'array vazio',
      assert: 'groupBy([],"c",rs=>rs.length).length === 0',
    },
  ]}
/>
