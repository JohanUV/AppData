## CASE WHEN e lógica condicional

`CASE WHEN` é o if/else do SQL. Permite gerar colunas condicionais,
categorizar dados, lidar com NULLs com elegância e mover lógica do app
para o warehouse.

### Duas sintaxes

**1. CASE buscado (mais usado):**

```sql
SELECT user_id, amount,
  CASE
    WHEN amount < 10 THEN 'small'
    WHEN amount < 100 THEN 'medium'
    WHEN amount < 1000 THEN 'large'
    ELSE 'huge'
  END AS bucket
FROM payments;
```

**2. CASE simples:**

```sql
CASE status
  WHEN 'paid' THEN 1
  WHEN 'pending' THEN 0
  ELSE -1
END
```

`ELSE` é opcional mas recomendado. Sem ele, sem match → NULL.

### Casos típicos

**Categorização:**

```sql
SELECT
  CASE
    WHEN age < 18 THEN 'minor'
    WHEN age < 30 THEN 'young_adult'
    WHEN age < 60 THEN 'adult'
    ELSE 'senior'
  END AS age_group,
  COUNT(*)
FROM users GROUP BY 1;
```

**Pivots manuais:**

```sql
SELECT
  date_trunc('day', created_at) AS day,
  SUM(CASE WHEN status = 'paid' THEN amount ELSE 0 END) AS revenue_paid,
  SUM(CASE WHEN status = 'refunded' THEN amount ELSE 0 END) AS revenue_refunded,
  SUM(CASE WHEN status = 'pending' THEN amount ELSE 0 END) AS revenue_pending
FROM payments GROUP BY 1;
```

**Contagens condicionais:**

```sql
SELECT user_id,
  COUNT(*) AS total_orders,
  COUNT(CASE WHEN status = 'paid' THEN 1 END) AS paid_orders
FROM payments GROUP BY user_id;
```

### CASE como NULL-handler

```sql
SELECT user_id, COALESCE(country, 'unknown') AS country FROM users;
```

`COALESCE(a, b, c)` retorna o primeiro não-NULL.

### IIF, IF, NULLIF

```sql
IIF(amount > 100, 'big', 'small')
NULLIF(quantity, 0)                -- evita divisão por zero
COALESCE(nickname, name, 'anon')
```

`NULLIF(x, y)` retorna NULL se x = y.

```sql
SELECT amount / NULLIF(quantity, 0) AS unit_price FROM order_items;
```

### Padrões avançados

**Tag first/last/only em partição:**

```sql
WITH ranked AS (
  SELECT *,
    ROW_NUMBER() OVER (PARTITION BY user_id ORDER BY created_at) AS rn,
    COUNT(*) OVER (PARTITION BY user_id) AS total
  FROM purchases
)
SELECT user_id, amount,
  CASE
    WHEN total = 1 THEN 'only'
    WHEN rn = 1 THEN 'first'
    WHEN rn = total THEN 'last'
    ELSE 'middle'
  END AS purchase_position
FROM ranked;
```

### Erros comuns

- **Esquecer `ELSE`** — NULL inesperado.
- **Misturar tipos nos THENs**.
- **`CASE` quando `COALESCE` ou `NULLIF` bastam**.
- **Aninhar 5 níveis**.
