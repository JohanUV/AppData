## ¿Qué es Data Engineering?

**Data Engineering** es la disciplina que se encarga de diseñar, construir y mantener
las infraestructuras y pipelines que mueven datos desde sus orígenes hasta los
sistemas analíticos donde se extrae valor.

A diferencia de un Data Scientist, que se enfoca en sacar conclusiones de los datos,
un Data Engineer construye los caminos por los que esos datos viajan: de manera
**confiable, escalable y reproducible**.

<Callout type="info" title="Idea clave">
  Sin un buen pipeline de datos, los modelos de Machine Learning más sofisticados
  no sirven de nada: basura entra, basura sale.
</Callout>

### El ciclo de vida del dato

<Diagram type="mermaid">
{`flowchart LR
  A[Fuentes] -->|Extracción| B[Staging]
  B -->|Transformación| C[Data Warehouse]
  C -->|Modelado| D[Marts]
  D -->|Consumo| E[BI / ML]`}
</Diagram>

### Stack típico

Hoy en día un equipo de Data Engineering suele combinar:

- **Orquestación**: Airflow, Dagster, Prefect.
- **Procesamiento batch**: Spark, dbt, BigQuery.
- **Procesamiento streaming**: Kafka, Flink, Kinesis.
- **Almacenamiento**: S3, GCS, HDFS, Iceberg.
- **Lenguajes**: Python y SQL son la dupla universal.

<CodeBlock language="sql" filename="ejemplo.sql">
{`-- Una transformación SQL típica en un Data Warehouse
SELECT
  date_trunc('day', event_at)        AS day,
  count(distinct user_id)            AS dau,
  sum(amount)                        AS revenue
FROM events.purchases
WHERE event_at >= current_date - 30
GROUP BY 1
ORDER BY 1;`}
</CodeBlock>

<Callout type="tip" title="Recuerda">
  El 80% del tiempo de un Data Engineer se va en mover, limpiar y validar datos.
  El 20% restante en la lógica de negocio. Aceptarlo es el primer paso.
</Callout>

<Exercise

  title="Suma de columnas"
  description="Implementa una función `sumColumn(rows, column)` que sume los valores de una columna numérica en un array de objetos."
  language="javascript"
  starterCode={`function sumColumn(rows, column) {
  // Tu código aquí
  return 0;
}

// No modifiques las siguientes líneas:
const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
  solution={`function sumColumn(rows, column) {
  return rows.reduce((acc, r) => acc + (r[column] ?? 0), 0);
}

const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
