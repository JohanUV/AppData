## What is Data Engineering?

**Data Engineering** is the discipline of designing, building and maintaining the
infrastructure and pipelines that move data from its sources to the analytical
systems where value is extracted.

Unlike a Data Scientist, who focuses on drawing conclusions from data, a Data
Engineer builds the roads the data travels on: **reliably, at scale, and reproducibly**.

<Callout type="info" title="Key idea">
  Without a solid data pipeline, the most sophisticated ML models are useless.
  Garbage in, garbage out.
</Callout>

### The data lifecycle

<Diagram type="mermaid">
{`flowchart LR
  A[Sources] -->|Extract| B[Staging]
  B -->|Transform| C[Data Warehouse]
  C -->|Modeling| D[Marts]
  D -->|Consume| E[BI / ML]`}
</Diagram>

### Typical stack

A Data Engineering team today usually combines:

- **Orchestration**: Airflow, Dagster, Prefect.
- **Batch processing**: Spark, dbt, BigQuery.
- **Stream processing**: Kafka, Flink, Kinesis.
- **Storage**: S3, GCS, HDFS, Iceberg.
- **Languages**: Python and SQL are the universal duo.

<CodeBlock language="sql" filename="example.sql">
{`-- A typical SQL transformation in a Data Warehouse
SELECT
  date_trunc('day', event_at)        AS day,
  count(distinct user_id)            AS dau,
  sum(amount)                        AS revenue
FROM events.purchases
WHERE event_at >= current_date - 30
GROUP BY 1
ORDER BY 1;`}
</CodeBlock>

<Callout type="tip" title="Remember">
  80% of a Data Engineer's time is moving, cleaning and validating data. The
  remaining 20% is business logic. Accepting this is step one.
</Callout>

## Hands-on exercise

<Exercise
  id="intro-exercise-sum"
  title="Column sum"
  description="Implement `sumColumn(rows, column)` that sums the values of a numeric column in an array of objects."
  language="javascript"
  starterCode={`function sumColumn(rows, column) {
  // Your code here
  return 0;
}

// Don't modify the following lines:
const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
  solution={`function sumColumn(rows, column) {
  return rows.reduce((acc, r) => acc + (r[column] ?? 0), 0);
}

const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
  hints={[
    'Walk the array and accumulate the values. Array.prototype.reduce works well here.',
    'Handle the case where the column may be missing (use ?? 0).',
  ]}
  tests={[

    {
      description: 'sumColumn returns 0 for an empty array',
      assert: 'sumColumn([], "amount") === 0',
    },
    {

      assert: 'sumColumn([{a:1},{a:null},{a:2}], "a") === 3',
