## O que é Data Engineering?

**Data Engineering** é a disciplina que projeta, constrói e mantém a
infraestrutura e os pipelines que movem dados das suas origens aos sistemas
analíticos onde o valor é extraído.

Diferente de um Data Scientist, focado em tirar conclusões dos dados, um Data
Engineer constrói os caminhos pelos quais os dados viajam: de forma
**confiável, escalável e reproduzível**.

<Callout type="info" title="Ideia chave">
  Sem um pipeline sólido, os modelos de ML mais sofisticados são inúteis.
  Lixo entra, lixo sai.
</Callout>

### Stack típica

Hoje uma equipe de Data Engineering geralmente combina:

- **Orquestração**: Airflow, Dagster, Prefect.
- **Processamento batch**: Spark, dbt, BigQuery.
- **Processamento streaming**: Kafka, Flink, Kinesis.
- **Armazenamento**: S3, GCS, HDFS, Iceberg.
- **Linguagens**: Python e SQL são a dupla universal.

<CodeBlock language="sql" filename="exemplo.sql">
{`-- Uma transformação SQL típica em um Data Warehouse
SELECT
  date_trunc('day', event_at)        AS day,
  count(distinct user_id)            AS dau,
  sum(amount)                        AS revenue
FROM events.purchases
WHERE event_at >= current_date - 30
GROUP BY 1
ORDER BY 1;`}
</CodeBlock>

<Callout type="tip" title="Lembre-se">
  80% do tempo de um Data Engineer vai em mover, limpar e validar dados.
  Os 20% restantes em lógica de negócio. Aceitar isso é o primeiro passo.
</Callout>

## Exercício prático

<Exercise
  id="intro-exercise-sum"
  title="Soma de coluna"
  description="Implemente `sumColumn(rows, column)` que soma os valores de uma coluna numérica em um array de objetos."
  language="javascript"
  starterCode={`function sumColumn(rows, column) {
  // Seu código aqui
  return 0;
}

// Não modifique as linhas a seguir:
const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
  solution={`function sumColumn(rows, column) {
  return rows.reduce((acc, r) => acc + (r[column] ?? 0), 0);
}

const data = [
  { user: 'a', amount: 10 },
  { user: 'b', amount: 25 },
  { user: 'c', amount: 7 },
];
const answer = sumColumn(data, 'amount');
console.log('total:', answer);`}
  hints={[
    'Percorra o array e acumule os valores. Array.prototype.reduce funciona bem.',
    'Trate o caso em que a coluna pode não existir (use ?? 0).',
  ]}
  tests={[
    { description: 'answer === 42 para os dados de demo', assert: 'answer === 42' },
    {
      description: 'sumColumn devolve 0 com array vazio',
      assert: 'sumColumn([], "amount") === 0',
    },
    {
      description: 'ignora valores nulos na coluna',
      assert: 'sumColumn([{a:1},{a:null},{a:2}], "a") === 3',
    },
  ]}
